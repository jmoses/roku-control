# Roku

Gem to control a roku device

## Installation

Add this line to your application's Gemfile:

    gem 'roku'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roku

## Usage

Just look at remote.rb

It's a working example.

    $ ruby remote.rb IP_OF_ROKU
